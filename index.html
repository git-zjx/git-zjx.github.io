Linux-环境变量

2018-05-21
Linux

概念

一般是指在操作系统中用来指定操作系统运行环境的一些参数

类型

  • 全局变量
  • 本地变量

全局变量

在 shell 会话和该 shell 产生的任何子进程中都可见
系统环境变量始终使用全大写字母来区别于普通用户环境变量

查看

printenv 查看所有全局变量
echo $HOME 查看指定全局变量

创建

先创建一个本地环境变量，然后再将它导出到全局环境中

移除

在子进程中移除全局变量，父进程中仍能使用

本地变量

只在创建它们的 shell 中可见

查看

没有可以只显示本地环境变量的命令，set命令会显示特定进程的所有环境变量集，但其中也包括了全局环境变量

创建

创建新的环境变量，建议使用小写字母，方便区分个人环境变量与系统环境变量

移除

默认的环境变量

bash shell Bourne 变量
| 变量 | 描述 |
| :-: | :-: |
| CDPATH | 冒号分隔的目录列标，用作 cd 命令的搜索路径 |
| HOME | 当前用户的主目录 |
| IFS | 用于分隔字段的字符列标，shell 使用它们分隔文本字符串 |
| MAIL | 当前用户邮箱的文件名。对于新邮件，bash shell 将检查该文件 |
| MAILPATH | 当前用户邮箱的多个文件名，由冒号分隔。对于新邮件，bash shell 将检查该列表中的各个文件 |
| OPTARG | getopts 命令处理的最后一个选项参数的值 |
| OPTIND | getopts 命令处理的最后一个选项参数的索引值 |
| PATH | 冒号分隔的目录列标，shell 将在这些目录中查找命令 |
| PS1 | 主 shell 命令行界面提示字符串 |
| PS2 | 次 shell 命令行界面提示字符串 |
