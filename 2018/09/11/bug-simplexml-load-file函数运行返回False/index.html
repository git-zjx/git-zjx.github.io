bug-simplexml_load_file函数运行返回False

2018-09-11
bug

文章目录

  1. 来源
  2. 思路
  3. 解决方案
  4. 参考

来源

阿里云主机上运行 simplexml_load_file函数运行返回False 大部分情况下会出现获取不到返回值的情况，目前没有找到根源，手头的两个云主机，一个可以正常运行，一个就会出现问题

思路

排除文件权限问题

排除了 php.ini 配置 disable_functions

cli 模式下可用，且是同一 php.ini，排除配置文件的问题

解决方案

使用 file_get_contents 和 simplexml_load_string 替换 simplexml_load_file函数运行返回False
