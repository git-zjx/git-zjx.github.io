基本脚本函数

创建函数

使用函数

函数名 func1 每次被引用时，bash shell 返回 func1 函数的定义并执行该函数定义的全部命令，函数使用须在函数定义之后

返回值

bash shell 将函数看作小型脚本，并以退出状态结束

默认退出状态

默认情况下，函数的退出状态是函数的最后一条命令返回的退出状态。函数执行完毕之后，可以使用标准变量 $? 来确定函数的退出状态

默认退出状态下，无法确定该函数的其他命令是否执行成功

使用 return 命令

return 命令可以使用单个整数值来定义函数的退出状态

有两点需要注意：

  • 函数完成后尽快提取返回值，变量 $? 返回的是最近已执行命令的退出状态
  • 退出状态的取值范围是 0-255，数据大于 256 将返回错误值

使用函数输出

函数的输出可以捕获并存放在 shell 变量中，以此可以从函数获取任意类型的输出并给变量赋值

在函数中使用变量

向函数传递参数

函数可以使用标准参数环境变量来表示命令行传递给函数的参数。例如：函数名在变量 $0 中定义，函数命令行的其他参数使用变量 $1 和 $2 等定义。专用变量 $# 可以用来确定传递给函数的参数数目

函数无法从脚本命令行直接访问脚本参数，须在调用函数时手动传递

在函数中处理变量

全局变量

默认情况下，脚本中定义的变量都是全局变量。

局部变量

函数内部使用的变量可以成为局部变量，需要在声明前冠以 local 关键字

数组变量与函数

向函数传递数组

如果将数组变量作为函数参数使用，那么函数只会提取数组变量的第一个取值

可以将数组拆分为单个元素，然后使用这些元素值作为函数参数，函数内部可以再将这些参数重组为新数组变量

从函数返回数组

函数使用 echo 语句以恰当的顺序输出数组各元素的值，然后脚本将这些数据重组为新数组变量

函数递归

自给是局部函数变量的一个特性。自给函数怵了脚本通过命令行传递的变量，不使用函数之外的任何资源，这个特性可以使函数能够递归调用

创建库

bash shell 可以创建函数的库文件，然后在不同脚本中引用

使用 source 命令可以在 shell 脚本内部运行库文件脚本，它有一个短小的别名，成为点操作符

在命令行中使用函数

在命令行创建函数

命令行中创建的函数，可以在任意目录使用

如果自定义函数与内置命令或其他命令同名，那么这个自定义函数将覆盖原有命令

在 .bashrc 文件中定义函数

在 .bashrc 文件中定义函数可以避免每次退出 shell 后函数失效

直接定义函数

可以直接在 .bashrc 文件末尾添加自定义函数

提供函数文件

可以使用 source 命令将现有库文件的函数包含进 .bashrc 脚本

shell 会将全部已定义的函数传递给子 shell 进程，对于所有从 shell 会话运行的脚本，自定义函数也可用
