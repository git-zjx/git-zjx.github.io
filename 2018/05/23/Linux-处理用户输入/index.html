    • 命令行参数

    读取参数

    bash shell 将在命令行输入的所有参数赋值给一些特殊变量，这些变量称为位置参数。位置参数通过标准数字表示，其中 $0 为程序名称，$1 为第一个参数，$2 为第二个参数，依次类推，直到 $9 为第九个参数。如果脚本需要的命令行参数多于 9 个，那么之后的变量须使用大括号括起来，如 ${10}。

    参数在命令行中使用空格分隔，如果参数中包含空格，必须使用引号

    读取程序名称

    使用参数 $0 可以确定 shell 从命令行启动的程序的名称。但传递给变量 $0 的字符串实际上是程序的完整路径，而不仅仅是程序名

    可以通过使用 basename 命令只返回程序名

    特殊的参数变量

    参数计数

    特殊变量 $# 中存储执行脚本时包含的命令行参数的个数

    可以通过 ${!#} 来获取最后一个命令行的参数值，因为不能在大括号中使用 $ 符号，必须用 ! 进行替换

    获取所有数据

    变量 $* 将命令行中提供的所有参数作为一个单词处理。这个单词包含出现在命令行中的每一个参数值。本质上，变量 $* 不是将参数视为多个对象，而是将它们看作一个参数

    变量 $@ 将命令行中提供的所有参数作为同一个字符串中的多个单词处理。允许对其中的值进行迭代，分隔开所提供的不同参数

    移位

    shift 命令默认将每个参数变量左移一个位置。变量 $3 的值给变量 $2，变量 $2 的值给 $1，而变量 $1 的值被丢弃，$0 变量始终保持不变

    参数移位掉之后无法恢复，须谨慎

    使用 getopt 命令

    getopt 命令可以接受任意形式的命令行选项和参数列表，并自动将这些选项和参数转换成适当的格式。命令格式如下：

    选项字符串（optstring）定义命令行中有效选项的字母，还定义哪些选项需要参数值

    其中选择字符串定义了四个有效选项字母，a、b、c 和 d。还定义选项字母 b 需要一个参数值（选项字母后跟一个冒号）。当执行 getopt 命令时，会检测提供的参数列表，然后基于提供的选项字符串对列表进行解析。解析时自动将 -cd 选项分隔成两个不同的选项，并插入双破折号来分隔行中的额外参数

    -q 选项可以忽略因为指定选项不包含在选项字符串中出现的错误信息

    将原始脚本命令行参数送给 getopt 命令，然后将 getopt 命令的输出送给 set 命令，以便将原始命令行参数替换为通过 getopt 格式化过的更精细的形式

    getopt 命令不能很好的处理带有空格的参数值，它将空格解析为参数分隔符，而不是将双引号引起来的两个值合并为一个参数

    getopts 命令

    getopts 命令顺序的对现有的 shell 参数变量进行处理。每调用一次 getopts，它值处理在命令中检测到的参数中的一个。处理完所有参数后，以大于零的退出状态退出。

    与 getopt 命令使用相似，在选项字符串中列出有效选项字母，如果选项字母需要参数值后面加上一个冒号。如果要禁止输出错误信息，那么使选项字符串以冒号开头。getopts 命令将当前参数放在命令行中定义的 variable 中

    getopts 命令使用两个环境变量。环境变量 OPTARG 中包含需要参数值的选项要使用的值。环境变量 OPTIND 包含的值表示 getopts 停止处理时在参数列表中的位置。这样，处理完选项后可以继续处理其他命令行参数

    当 getopts 命令解析命令行选项时，会将打头的破折号去掉，所以 case 定义中不需要破折号

    选项字母和参数值中间可以没有空格

    会将在命令行中找到的未定义的选项都绑定为单一的问号输出

    getopts 每处理一个选项，环境变量 OPTIND 的值会增加 1，当达到 getopts 处理的末尾时，可以使用 shift 命令和 OPTIND 值进行操作来移动到参数

    获取用户输入

    基本读取

    read 命令接受标准输入（键盘）的输入，或其他文件描述符的输入。得到输入后，read 命令将数据放入一个标准变量中

    read 命令的 -p 选项，允许在 read 命令行中直接指定一个提示

    read 命令将提示后面输入的所有数据赋值个单个变量，或者也可以指定多个变量。输入的每个数据值都被赋值给列表中的下一个变量。如果变量列表中变量数目少于数据数目，则剩余数据都赋值个最后一个变量

    也可以不指定变量。如果不指定变量，那么 read 命令会将接收到的数据放置在环境变量 REPLY 中

    计时

    -t 选项指定 read 命令待输入的秒数，当计时器计时数满时，read 命令返回一个非零退出状态

    还可以设置 read 命令计数输入的字符。当输入的字符达到预定数目时自动退出，并将输入的数据赋值给变量，使用 -n1 表示只要接收到一个字符就退出

    cat test22

    #!/bin/bash

    read -s -p “Enter your password: “ pass

    echo
    echo “Is your password really $pass ?

    ./test22

    Enter your password:
    Is your password really qqqwwweee ?
