if-then

if 语句运行在 if 行定义的命令，如果命令的退出状态是 0，将执行 then 后面的所有命令。如果命令的退出状态是 0 以外的其他值，那么 then 后面的命令不执行

if-then-else

if 语句运行在 if 行定义的命令，如果命令的退出状态是 0，将执行 then 后面的所有命令。如果命令的退出状态是 0 以外的其他值，将执行 else 后面的所有命令

嵌套 if 语句

bash shell 会按顺序执行 if 语句，只有第一个返回 0 退出状态的命令会导致 then 部分命令被执行

test 命令

test 命令提供一种检测 if-then 语句不同条件的方法。如果 test 命令中列出的条件评估值为 true，test 命令以 0 退出状态码退出

数值比较

比较 描述
n1 -eq n2 检查 n1 是否等于 n2
n1 -ge n2 检查 n1 是否大于或等于 n2
n1 -gt n2 检查 n1 是否大于 n2
n1 -le n2 检查 n1 是否小于或等于 n2
n1 -lt n2 检查 n1 是否小于 n2
n1 -ne n2 检查 n1 是否不等于 n2

只能比较整数数字

字符串比较

比较 描述
str1 = str2 检查 str1 与 str2 是否相同
str1 != str2 检查 str1 与 str2 是否不同
str1 < str2 检查 str1 是否小于 str2
str1 > str2 检查 str1 是否大于 str2
-n str1 检查 str1 的长度是否大于 0
-z str1 检查 str1 的长度是否为 0

比较大小时需要注意：

  • 大于和小于符号需要转义
  • 大于和小于的顺序与在 sort 命令中的顺序不同，test 命令使用标准的 ASCII 排序，sort 命令使用为系统当前语言设置定义的排列顺序

文件比较

比较 描述
-d file 检查 file 是否存在并且是一个目录
-e file 检查 file 是否存在
-f file 检查 file 是否存在并且是一个文件
-r file 检查 file 是否存在并且可读
-s file 检查 file 是否存在不为空
-w file 检查 file 是否存在并且可写
-x file 检查 file 是否存在并且可执行
-O file 检查 file 是否存在并且被当前用户拥有
-G file 检查 file 是否存在并且默认组是否为当前用户组
file1 -nt file2 检查 file1 是否比 file2 新
file1 -ot file2 检查 file1 是否比 file2 旧

高级特征

双圆括号

双圆括号允许在比较中包含高级数据公式

比较 描述 比较 描述
val++ 后增量 << 逐位左移
val-- 后减量 >> 逐位右移
++val 前增量 & 逐位布尔逻辑与
--val 前减量 &#124; 逐位布尔逻辑或
! 逻辑否定 && 逻辑与
~ 逐位取反 &#124;&#124; 逻辑或
** 取幂

双方括号

双方括号为字符串比较提供高级功能

使用在 test 命令中使用的标准字符串比较，同时支持模式匹配。

模式匹配中，可以定义与字符串值相匹配的正则表达式

case

case 命令将指定的变量与不同的模式进行比较。如果变量与模式匹配，shell 执行为该模式指定的命令。可以在一行中列出多个模式，使用竖条操作符将每个模式分开。星号是与任何列出的模式都不匹配的值
