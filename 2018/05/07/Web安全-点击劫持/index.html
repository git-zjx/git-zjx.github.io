Web安全-点击劫持

2018-05-07
Web安全

概念

点击劫持是一种视觉上的欺骗手段，攻击者使用一个透明的、不可见的 iframe ，覆盖在一个网页上，然后诱使用户在该网页上进行操作

攻击类型

  • iframe 攻击
  • Flash 攻击
  • 图片覆盖攻击
  • 拖拽攻击（利用浏览器的拖拽事件）
  • 触屏劫持

防御

frame busting

使用 Javascript 代码禁止 iframe 嵌套

X-Frame-Options

X-Frame-Options 有三个可选值：DENY、SAMEORIGIN、ALLOW-FROM origin

当值为 DENY 时，浏览器会拒绝当前页面加载任何 frame 页面
当值为 SAMEORIGIN 时，浏览器只加载地址为同源域名下的 frame 页面
当值为 ALLOW-FROM origin 时，浏览器会加载所有 frame 页面
